# 雷达检测工具

用于检测雷达点云频率和点云总数，判断雷达工作状态的工具集。

## 🎯 功能特性

- **点云频率检测**: 实时监控雷达点云数据的发布频率
- **点云数量统计**: 统计每帧点云的点数，计算平均值
- **性能评估**: 根据预设阈值判断雷达工作状态
- **详细报告**: 生成包含统计数据的详细检测报告
- **实时监控**: 检测过程中实时显示状态信息
- **灵活配置**: 支持自定义检测时长、频率阈值、点数阈值等参数

## 📁 文件结构

```
lidar_tools/

```

## 🚀 快速开始

### 方法1: 使用启动脚本（推荐）

```bash
# 进入工具目录
cd tools/checktools/lidar_tools

# 设置执行权限
chmod +x run_lidar_test.sh

# 使用默认参数运行（检测30秒）
./run_lidar_test.sh

# 自定义参数运行
./run_lidar_test.sh -d 60 -f 10 -p 2000
```

### 方法2: 直接运行Python脚本

```bash
# 进入工具目录
cd tools/checktools/lidar_tools

# 使用默认参数
python3 lidar_test_tool.py

# 自定义参数
python3 lidar_test_tool.py --duration 60 --min-frequency 10 --min-point-count 2000
```

## ⚙️ 参数说明

### 启动脚本参数

| 参数 | 短参数 | 说明 | 默认值 |
|------|--------|------|--------|
| `--duration` | `-d` | 检测持续时间（秒） | 30 |
| `--frequency` | `-f` | 最小点云频率阈值（Hz） | 5.0 |
| `--points` | `-p` | 最小点云总数阈值 | 1000 |
| `--topic` | `-t` | 点云话题名称 | `/velodyne_points` |
| `--help` | `-h` | 显示帮助信息 | - |

### Python脚本参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--duration` | 检测持续时间（秒） | 30 |
| `--min-frequency` | 最小点云频率阈值（Hz） | 5.0 |
| `--min-point-count` | 最小点云总数阈值 | 1000 |
| `--topic` | 点云话题名称 | `/velodyne_points` |

## 📊 检测指标

### 频率检测
- **平均频率**: 检测期间的平均点云发布频率
- **频率稳定性**: 帧间时间间隔的标准差，越小越稳定
- **最小/最大间隔**: 帧间时间间隔的极值

### 点云数量检测
- **总点数**: 检测期间接收到的所有点云点数总和
- **平均每帧点数**: 每帧点云的平均点数
- **点数稳定性**: 各帧点数的变化情况



## 📈 使用示例

### 基础检测
```bash
# 检测30秒，使用默认阈值
./run_lidar_test.sh
```

### 长时间检测
```bash
# 检测2分钟，频率阈值8Hz，点数阈值50000
./run_lidar_test.sh -d 120 -f 8 -p 50000
```

### 自定义话题
```bash
# 检测自定义话题的点云数据
./run_lidar_test.sh --topic /rslidar_points
```

### 高精度检测
```bash
# 检测1分钟，高频率阈值，高点数阈值
./run_lidar_test.sh -d 60 -f 15 -p 100000
```
