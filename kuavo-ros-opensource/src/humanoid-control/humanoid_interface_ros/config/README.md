# 转向区间配置文件说明

## 文件位置
`config/step_turn.yaml`

## 功能说明
此配置文件用于设置机器人单步转向控制的区间参数。系统会根据摇杆的输入值，自动选择对应的转向区间和轨迹。

## 配置原理

### 对称性设计
- **只需配置左转区间**：参数文件中只需要设置左转的参数
- **自动生成右转**：代码会自动根据左转参数生成对称的右转区间

### 对称规则
左转区间配置后，右转区间按以下规则自动生成：
- **摇杆区间范围**：取反（负值变正值）
  - 左转: `-0.15 ~ 0.0` → 右转: `0.0 ~ 0.15`
- **body_x**：保持不变
- **body_y**：取反（正变负，负变正）
- **body_z**：保持不变
- **body_yaw**：取反（正变负，负变正）

## 参数说明

### turn_zones_count
转向区间的数量（只计算左转区间数量，右转会自动生成）

### left_turn_zones
左转区间配置列表，每个区间包含以下参数：

| 参数名 | 类型 | 说明 |
|--------|------|------|
| `min_value` | float | 摇杆区间最小值（负值，范围 -1.0 ~ 0.0） |
| `max_value` | float | 摇杆区间最大值（负值，范围 -1.0 ~ 0.0） |
| `body_x` | double | 身体X方向位移（米） |
| `body_y` | double | 身体Y方向位移（米，正值表示左转） |
| `body_z` | double | 身体Z方向位移（米） |
| `body_yaw` | double | 身体偏航角（度，正值表示左转） |

## 配置示例

```yaml
turn_zones_count: 4

left_turn_zones:
  # 小角度左转
  - min_value: -0.15
    max_value: 0.0
    body_x: -0.01
    body_y: 0.01      # 右转时自动变为 -0.01
    body_z: 0.0
    body_yaw: 15.0    # 右转时自动变为 -15.0
  
  # 中角度左转
  - min_value: -0.45
    max_value: -0.15
    body_x: -0.03
    body_y: 0.03
    body_z: 0.0
    body_yaw: 30.0
```

## 生成结果示例

上述配置会生成以下转向区间：

**右转区间**（自动生成）：
- 区间0: `0.0 ~ 0.15`，pose=`[-0.01, -0.01, 0.0, -15.0]`
- 区间1: `0.15 ~ 0.45`，pose=`[-0.03, -0.03, 0.0, -30.0]`

**左转区间**（从配置文件读取）：
- 区间0: `-0.15 ~ 0.0`，pose=`[-0.01, 0.01, 0.0, 15.0]`
- 区间1: `-0.45 ~ -0.15`，pose=`[-0.03, 0.03, 0.0, 30.0]`

## 修改配置

1. 编辑 `config/step_turn.yaml` 文件
2. 只需修改左转区间的参数
3. 重启节点，配置会自动加载
4. 如果 YAML 文件加载失败，系统会使用默认配置

## 注意事项

1. **区间连续性**：建议配置的区间相互衔接，避免出现空白区间
2. **摇杆范围**：左转区间应在 `-1.0 ~ 0.0` 范围内
3. **参数单位**：位移单位为米，角度单位为度
4. **备用方案**：如果 YAML 文件读取失败，系统会自动使用内置的默认配置
